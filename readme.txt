== Description ==

Better Member UX for MemberPress is a lightweight frontend enhancement plugin for MemberPress-powered membership sites.

It improves the default MemberPress account, login, and password reset experiences without editing MemberPress templates or requiring a page builder.

Features:

* Cleaner account page layout
* Improved account navigation
* Rename Home to Profile
* Optional Dashboard link
* Better profile form layout
* Improved subscription and payment tables
* Better mobile table layout
* Cleaner login page
* Cleaner password reset page
* Basic color and label settings

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/better-member-ux-for-memberpress/`, or install the plugin through the WordPress plugins screen.
2. Activate the plugin.
3. Go to Settings > Better Member UX.
4. Configure labels, colors, and the dashboard link if needed.

== Frequently Asked Questions ==

= Does this replace MemberPress templates? =

No. It enhances existing MemberPress frontend markup using CSS and JavaScript.

= Does this work with Elementor? =

Yes. It is designed to work with Elementor, Beaver Builder, Gutenberg, Classic Editor, Divi, and most themes because it targets MemberPress frontend markup.

= Does this change MemberPress form submissions? =

No. It does not alter form actions, nonces, fields, or submission handling.

= Can I disable individual enhancements? =

Yes. Account, login, and password reset enhancements can be enabled or disabled from the settings page.
