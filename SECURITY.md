# Security Policy

## Supported versions

Security fixes are applied to the latest released version.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's [private vulnerability reporting](https://github.com/ranaweb/better-member-ux-for-memberpress/security/advisories/new) instead.

Include reproduction steps, affected versions, potential impact, and a suggested mitigation when possible. Do not include real member credentials or personal data.
